# CSS структура

## Описание

CSS стили приложения организованы в модульную структуру для лучшей поддерживаемости.

## Структура

```
css/
├── base/              # Базовые стили
│   ├── variables.css  # CSS переменные
│   └── base.css      # Базовые стили
├── styles.css        # Основной файл со всеми стилями (извлечен из index.html)
└── main.css         # Главный файл для импортов (заготовка)
```

## Использование

Основной файл стилей подключен в `index.html`:
```html
<link rel="stylesheet" href="css/styles.css" />
```

## Примечания

- `styles.css` содержит все стили, извлеченные из `index.html`
- В будущем можно разделить `styles.css` на модули по компонентам
- `main.css` можно использовать для импорта модулей через `@import`
